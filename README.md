# 📊 Sales Analytics Dashboard

An interactive Sales Analytics Dashboard built using Excel, Python, and Streamlit to analyze business performance and generate insights.

---

## 🚀 Live Demo
👉 https://sales-analytics-dashboard-4ajutpwfm4apphwtobdqkpv.streamlit.app/

---

## 💻 GitHub Repository
👉 https://github.com/geetha-sandhya/sales-analytics-dashboard

---

## 🧠 Features
- Data cleaning and preprocessing using pandas
- Sales analysis by category and region
- KPI metrics (Total Sales, Top Category, Lowest Category)
- Interactive dashboard using Streamlit
- Visualizations for better decision-making

---

## 🛠️ Technologies Used
- Python
- Pandas
- Streamlit
- Excel

---

## 📊 Insights
- Technology category generates highest revenue
- Office Supplies is the lowest performing category
- Sales vary significantly across categories

---

## 📌 Project Goal
To analyze sales data and provide actionable insights through a user-friendly dashboard.
