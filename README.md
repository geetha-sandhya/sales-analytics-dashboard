# 📊 AI-Enhanced Sales Analytics Dashboard

## 📌 Project Overview
This project focuses on analyzing sales data to extract meaningful business insights and present them through an interactive dashboard. The solution combines Excel, Python, and Streamlit to perform end-to-end data analysis and visualization.

---
 

## 🎯 Objective
The main objective of this project is to:
- Analyze sales performance across categories and regions
- Identify key trends and business insights
- Build an interactive dashboard for easy data exploration
- Deploy the solution as a live web application

---

## 📂 Dataset
The project uses the *Superstore Sales Dataset*, which contains:
- Order details (Order ID, Order Date)
- Product information (Category, Sub-category, Product Name)
- Regional data (Region)
- Sales values

---

## 🧹 Step 1: Data Cleaning (Excel)
- Verified data consistency
- Handled missing values
- Ensured proper formatting of columns (dates, numbers)

---

## 📊 Step 2: Data Analysis (Excel Pivot Tables)
- Created pivot tables to summarize sales data
- Analyzed total sales by category and region
- Identified key performance metrics

---

## 📈 Step 3: KPI Creation
Key Performance Indicators (KPIs) were generated:
- *Total Sales:* 2,261,536
- *Top Category:* Technology
- *Lowest Category:* Office Supplies

These KPIs help in quickly understanding business performance.

---

## 📉 Step 4: Data Visualization (Excel)
- Created bar charts for sales by category
- Improved dashboard layout for better readability
- Highlighted key insights visually

---

## 🐍 Step 5: Data Analysis using Python
Used Python for automation and deeper analysis:
- Library used: pandas
- Loaded dataset using CSV
- Calculated total sales and category-wise performance
- Automated insights generation

---

## 🌐 Step 6: Dashboard Development (Streamlit)
- Built an interactive dashboard using Streamlit
- Displayed KPIs and charts
- Enabled user-friendly data visualization

---

## 🚀 Step 7: Deployment
- Deployed the application using Streamlit Cloud
- Converted the project into a live web application
